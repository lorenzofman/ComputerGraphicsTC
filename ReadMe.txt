Requisitos básicos implementados:

Insirção de pontos de controle
Translação dos pontos de controle com o uso do mouse
Visualização das blending functions
Visualização do grafo de controle. O fecho convexo conta como bônus.
Visualização da curva gerada
Visualização da animação da curva sendo gerada. 
Controle de FPS

Bônus:
Editor de curvas cúbicas com continuidade c1
Caso o usuário mova um ponto. Outros pontos são movidos junto para manter a continuidade
Curva é desenhada com thickness para ficar mais bonita
Interface similar a outros softwares de manipulação de Bezier:
    - Anchor points são os pontos onde a curva passa
    - Tangent points são aqueles que 'orientam' a curva

Como usar:
Tudo pode ser feito através da interface com o mouse. Ao clicar na tela um novo Anchor point e seus pontos auxiliares são criados automaticamente.
 Animações são tocadas automaticamente após inserção de novo ponto.
Todas animações vem habilitadas, mas se o usuário quiser que a interface não fique poluída pode desabilitar usando:
C - Habilitar/Desabilitar desenho de grafo de construção
F - Habilitar/Desabilitar desenho do gráfico das blending functions
G - Habilitar/Desabilitar desenho do grafo de controle

Além disso o usuário pode tocar a animação novamente pressionando 'Espaço'
